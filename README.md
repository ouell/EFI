|       Type       | Item                                   |
|:----------------:|----------------------------------------|

| **Motherboard**  | [ Ga-Ab350m-Ds3h V2 ]    |
|     **RAM**      | [  16GB X2 3000Mhz ]   |
|     **GPU**      | [  AMD Sapphire Pulse Radeon 6600 ] |
|     **SSD**      | [ Asgard NVME 1TB ]  |
| **Power Supply** | [ Fonte Gamemax GM500 500W, 80 Plus Bronze ]  |
|                  |                                        |
|    **SMBIOS**    | [ MacPro1,1 ]                          |
|    **MacOS**     | [ Monterey ]                           |
|   **Opencore**   | [ 0.8.0 ]                              |
